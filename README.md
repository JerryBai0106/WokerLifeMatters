## 项目介绍

Woker life matters 专注于各大公司作息统计
由于众所周知的原因，包含互联网在内的各类企业加班盛行且无监管。这导致劳动者入职前无法确定公司的具体工作时间。本项目依托于同学们的奉献，致力于统计各大公司的作息时间，为劳动者们提供准确的参考。
